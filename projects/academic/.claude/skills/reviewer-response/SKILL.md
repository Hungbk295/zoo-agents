## Steps
1. Đọc RD để biết reviewer comments, paper context
2. Categorize comments: major, minor, editorial
3. Draft response for each comment:
   - Acknowledge the point
   - Explain changes made (or why not)
   - Reference specific sections/pages
4. Cross-check: mỗi comment có response, mỗi change có evidence
5. Output: response letter + revised sections reference trong RD
