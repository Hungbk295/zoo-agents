## Steps
1. Đọc RD để biết synthesis goal, sources, output format
2. Read tất cả source materials
3. Extract key points per source
4. Identify patterns, contradictions, gaps
5. Write synthesis document với proper attribution
6. Output: structured document theo RD spec
