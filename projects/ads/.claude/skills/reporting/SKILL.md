## Steps
1. Đọc RD để biết report type, period, audience
2. Thu thập data từ sources (files, user input)
3. Tổng hợp metrics: spend, impressions, clicks, conversions, CPA, ROAS
4. So sánh period-over-period
5. Output: structured report với key takeaways
