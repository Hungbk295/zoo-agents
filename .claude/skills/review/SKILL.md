# Skill: review

Use before marking a task as completed.

1. Re-read the RD's Done Criteria — check each item.
2. Verify the output exists and is correct.
3. Look for obvious breakage or missing steps.
4. If any criteria is not met, note what's missing and keep status as `in_progress`.
5. If all criteria pass, proceed to wrap.

**Rule:** Don't mark done unless every checkbox in Done Criteria is checked.
